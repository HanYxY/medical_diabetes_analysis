# Dataset

This folder contains the dataset used in the diabetes analysis project.

File included:

diabetes.csv

The dataset includes medical measurements used to predict whether a patient has diabetes.

Key features:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome

Outcome indicates whether the patient has diabetes:

0 = No Diabetes  
1 = Diabetes
